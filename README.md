# Supply Chain Performance Dashboard

Data analysis project examining delivery performance across 180,519 real supply chain orders.

## Key Findings
- Overall late delivery rate of 54.8% — majority of orders arrive late
- Europe and Pacific Asia have the highest late delivery rates, both above average
- First and Second Class shipping surprisingly underperform vs Standard Class
- Fan Shop is the top revenue generating department

## KPIs Analyzed
- Total Sales: $36.7 million across all orders
- Average Profit per Order: $21.97
- Late Delivery Rate by Market
- Late Delivery Rate by Shipping Mode
- Sales by Department

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
DataCo Smart Supply Chain Dataset — 180,519 orders, 53 variables
Source: Kaggle

## Background
Built as part of a data analytics portfolio combining logistics domain knowledge 
with Python-based data analysis. The goal was to identify operational inefficiencies
in a real supply chain dataset — the kind of analysis directly applicable to 
logistics and procurement roles.
